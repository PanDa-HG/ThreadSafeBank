# ThreadSafeBank

ThreadSafeBank is a C++ CLI-based concurrent banking system built for learning low-level design and operating system concurrency concepts.

## Current Version

Version 1: Basic account operations

## Planned Features

- Account creation with 6-digit PIN
- Deposit, withdraw, balance check
- Transfer between accounts
- Transaction logging
- Account closing
- Race condition demonstration
- Mutex-based synchronization
- Custom reader-writer lock using mutex and condition_variable
- Deadlock-safe transfer
- Concurrent multi-transfer
- Benchmarking with multiple threads

## Tech Stack

- C++
- STL
- std::thread
- std::mutex
- std::condition_variable
- Git/GitHub
